Project Name: 3D Model Creator & Printing Service Integration

Description:

Our project is a web-based platform that enables users to easily create and modify 3D models using OpenSCAD, a parametric CAD software. The platform is designed for accessibility, targeting both beginners looking to learn 3D modeling and experienced users needing a simple interface for custom designs. Once a model is complete, users can seamlessly send it to a connected 3D printing service (like Slant3D) for affordable, high-quality prints.

Features:

  • Interactive 3D Modeling Interface: Integrates OpenSCAD, allowing users to edit parameters and see real-time 3D previews.
  • Pre-built Templates for Easy Customization: Users can start with ready-made templates and modify dimensions, shapes, and other parameters.
  • Export and Print Options: Users can export STL files or directly connect to a 3D printing service for an on-demand print option.
  • Tutorials and Educational Content: Step-by-step tutorials help users learn CAD principles and familiarize themselves with OpenSCAD.

Installation & Setup Instructions:

  1. Clone this repository.
  2. Install dependencies:
    3. npm install  # For frontend dependencies
pip install -r requirements.txt  # For backend dependencies
  3. Start the Angular frontend:
    4. ng serve
  4. Start the Flask backend:
    5. flask run
  5. Access the web app at http://localhost:4200.

Usage Instructions:

  1. Open the platform and select a template or start a new model.
  2. Adjust parameters as needed and preview your model.
  3. When satisfied, export the model or click "Send to Print" to connect with a 3D printing service.

API Documentation:

  • OpenSCAD Integration: Describes endpoints for generating previews and exporting STL files.
  • 3D Printing Service Integration: Outlines endpoints to submit designs to Slant3D (or similar services) for printing.

Future Improvements:

Blog Post Outline

Title:

Introducing the Ultimate 3D Modeling Platform for Beginners and Professionals Alike

Introduction:

Creating custom 3D models and bringing them to life can be intimidating for beginners, with many tools requiring extensive knowledge and expensive software. Our platform simplifies 3D modeling, offering a web-based, intuitive interface that integrates OpenSCAD for easy parameter-based design. Plus, with a built-in connection to a 3D printing service, users can take their designs from screen to reality without ever leaving the website.

Problem Statement:

Learning 3D modeling traditionally requires a lot of time and resources. Existing tools can be too complex for newcomers and often don’t have affordable, integrated printing solutions. We wanted to create a platform that’s both educational and practical, letting users learn the basics while also creating real, functional 3D models.

Solution and Key Features:

  • Accessible 3D Modeling: The platform uses OpenSCAD, an open-source CAD software that’s well-suited for parametric design, meaning users can control their model dimensions and shapes through easy-to-understand parameters.
  • Pre-built Templates: To simplify the process further, users can start with templates and modify only the essential dimensions and shapes, getting them comfortable with CAD principles while producing a usable 3D model.
  • Seamless 3D Printing: Through a printing service API, users can directly send their models to be printed. They get cost-effective access to physical models without having to download files and find external print services.

How It Works (Implementation):

  • Backend (Flask): Handles the OpenSCAD processing by executing scripts that generate STL files based on user-defined parameters.
  • Frontend (Angular): Provides an interactive UI, where users can adjust parameters, view live previews, and access tutorial content.
  • API for Printing Service: Connects with a 3D printing service (like Slant3D) to submit models for printing, streamlining the process of getting a physical object.

Challenges and Solutions:

  • OpenSCAD Integration: Running OpenSCAD in a way that’s compatible with our backend (especially on free hosting platforms) presented challenges, which we solved by using the AppImage for consistent results.
  • User Interface Design: Ensuring an intuitive UI for beginners was key. We used Angular Material and Bootstrap to rapidly prototype a clean, responsive design.
  • Connecting to the 3D Printing API: The API integration required adapting to the service’s requirements while ensuring smooth file handling and submission.

Benefits to Users:

  • Learn by Doing: Beginners can learn 3D modeling by experimenting with templates, watching tutorials, and seeing immediate results.
  • Save Time and Money: Users don’t need to purchase expensive software or hardware. They can design, export, and even print, all from one platform.
  • Accessible: The platform lowers the barriers for getting started with 3D modeling, making it easy for anyone to create functional models.

Conclusion:

Our platform offers a comprehensive solution for both beginners and experienced users to create, modify, and print 3D models with ease. By integrating OpenSCAD and connecting to a 3D printing service, we provide a seamless experience from design to physical object.
